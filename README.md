# Desafio Vitta

> Challenge for an internship at Vitta.

## Build Setup

``` bash

# Clone the repository
git clone https://github.com/vitormrts/desafio-vitta

# Go to the create repo
cd desafio-vitta

# Install dependencies
yarn

# Optimize the app
yarn build

# Run the app (port 8080)
yarn dev
```
